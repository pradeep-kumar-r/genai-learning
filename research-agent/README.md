# Research Agent

This is a simple example of an AI research assistant agent that uses a combination of an LLM & a vector db in an agentic workflow to answer questions about research papers.

## Prerequisites

- Python 3.10 or higher
- OpenAI API key
- Groq API key
- ChromaDB API key

## Setup

1. Clone the repository
2. Create a virtual environment
3. Install dependencies (requirements.txt)
4. Set environment variables (copy .env.example to .env and fill in the values)
5. Download the research paper
