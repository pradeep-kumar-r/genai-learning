# genai-learning
A collection of tutorials &amp; micro-projects meant as a learning pathway in GenAI (LLMs, RAG, Chains, Agentic AI - tools, memory etc.)

This list of tutorials is majorly inspired by a 4-week hands-on training summit organized in 2025 by ODSC (Open Data Science Conference). It also contains a few other tutorials implementing ideas I've got from other sources. 

#### Disclaimer: 
This list is not exhaustive and is meant to be a starting point for learning GenAI. 
It is not a comprehensive list of all the possible ways to use GenAI.
The purpose of this repo is purely for illustration purposes and is not meant to be used for any commercial benefit. 
It is not meant to be production ready. Use at your own risk.

## Tutorials

#### research-agent
A research assistant agent that uses a combination of an LLM & a vector db in an agentic workflow to answer questions about research papers from ArXiv. The following design elements are explored:
    - Vanilla LLM Augmented
    - Prompt Chaining
    - Routing 
    - Autonomous Agent

#### rag-sentiment-analysis
A simple example of using RAG to predict sentiment of AirBnB listings based on past guest reviews.
